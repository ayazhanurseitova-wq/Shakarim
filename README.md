# Қазақстан зиялылары: Шәкәрім Құдайбердіұлы

Бұл жоба ұлы қазақ ойшылы, ақыны және философы Шәкәрім Құдайбердіұлына арналған веб-сайт. Жоба мектеп/колледж тапсырмасы аясында жасалған.

## Дизайн және Технологиялар
* **Стиль:** Минималистикалық, жұмсақ түстер палитрасы.
* **Ерекшелік:** Барлық суреттер CSS (`filter: grayscale(100%);`) арқылы автоматты түрде ақ-қара түске ауысады.
* **Технологиялар:** HTML5, CSS3 (Flexbox/Grid), Vanilla JavaScript.
* **Қаріптер:** Google Fonts (`Lora` - мәтін үшін, `Nunito` - тақырыптар үшін).

## Сайт құрылымы
1. `index.html` — Басты бет (Тұлғамен таныстыру).
2. `about.html` — Өмірбаяны (Өмір жолы, Саяси қызметі, Түркістан идеясы).
3. `works.html` — Шығармашылығы (Тарихи мұрасы, еңбектері).
